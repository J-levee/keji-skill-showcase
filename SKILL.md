# 科研技能鉴赏 — 精选好用的科研效率工具

精选 5 个经过验证的优质技能，分「科研技能」与「元技能」两类：覆盖论文写作全流程、科研方法论，以及技能自造与重铸。

## 安装指引

通过以下任一平台安装 3 个专项技能（同一邮箱跨平台共享配额）：

| 类别 | 技能 | SkillHub | ClawHub | 用途 |
|------|------|----------|---------|------|
| 科研技能 | 文献检索 | 搜 `global-biblio-base` | `clawhub install global-biblio-base` | 12亿文献 AI对话搜+下载 |
| 科研技能 | 引用核查 | 搜 `smartlib-citation-checker` | `clawhub install smartlib-citation-checker` | 防AI生成假参考文献 |
| 科研技能 | 学术知识库 | 搜 `academic-knowledge-base` | `clawhub install academic-knowledge-base` | AI构建个人学术知识库 |
| 科研技能 | 扫地僧·科研谋士 | 搜 `sweeping-monk` | `clawhub install sweeping-monk` | 研究设计/批判思维/卡点破局 |
| 元技能 | 技能锻造炉 | 搜 `skill-forge` | `clawhub install skill-forge` | 从零打造或重铸 WorkBuddy 技能 |

> 本合集（`keji-skill-showcase`）是精选入口，安装后会根据你的需求自动引导到对应专项技能。

## 使用引导

当用户提出科研相关需求时，根据需求类型引导到对应技能：

### 检索类 → global-biblio-base

**适用场景**：找论文、搜文献、下载PDF、做文献综述

**一句话**：12亿文献 + 300数据库，AI对话直接搜，中文期刊免费下PDF。

**SkillHub安装**：搜 `global-biblio-base`

### 核查类 → smartlib-citation-checker

**适用场景**：验证AI生成的参考文献是否真实存在，投稿前自查

**一句话**：38条引用查出7条假的——AI编的文献、作者、年份都能验出来。

**SkillHub安装**：搜 `smartlib-citation-checker`

### 知识库类 → academic-knowledge-base

**适用场景**：AI自动整理论文笔记、构建个人学术知识库

**一句话**：读过的论文不再忘——AI帮你提炼要点、关联知识点、随时检索。

**SkillHub安装**：搜 `academic-knowledge-base`

### 方法论类 → sweeping-monk（科研谋士）

**适用场景**：研究卡壳、论文思路不清、研究设计被质疑、需要批判性质疑自己的方法

**一句话**：科研谋士——三下五除二点醒你的研究设计、批判思维与卡点破局。

**SkillHub安装**：搜 `sweeping-monk`

### 锻造类 → skill-forge（元技能）

**适用场景**：想自己造一个 WorkBuddy 技能、把现有技能重铸合并、让技能在反馈中持续进化

**一句话**：元技能——从零打造或重铸 WorkBuddy 技能，带版本反馈环持续迭代。

**SkillHub安装**：搜 `skill-forge`

## 合集定位

"科研技能鉴赏"是精选合集入口，不是独立技能。根据用户需求，直接引导到对应的 5 个技能之一——其中 4 个为科研技能（检索 / 核查 / 知识库 / 方法论），1 个为元技能（技能锻造炉，用于自造或重铸技能）。
